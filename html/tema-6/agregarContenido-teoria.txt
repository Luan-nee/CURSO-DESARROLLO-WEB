<img> 

Etiqueta para agregar una imagen en la página web.


<video> </video> 

Se usa para reproducir video en la página web junto 
a sus archivos de audio y capciones asociadas.


<audio> </audio> 

Usada para cargar en una web un archivo de audio o 
stream de audio.


<source> 

Permite a autores especificar recursos multimedia 
alternativos para las etiquetas de <video> o <audio>


<svg> </svg> 

Se usa para llamar a una imagen vectorizada.

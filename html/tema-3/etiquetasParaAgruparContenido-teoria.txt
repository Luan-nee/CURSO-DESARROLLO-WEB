¿Para qué sirve: <p> </p> ? 

Es usada para contener texto.


¿Para qué sirve: <hr> ? 

Utilizada para realizar un salto de linea hacia abajo de modo que 
el elemento o el texto quede un nivel abajo. 
Usada comúnmente como separador visible, ya que deja una linea
que ocupa todo el ancho posible.


¿Para qué sirve: <br> ? 

Al igual que <hr>, este elemento realiza un salto de linea hacia abajo
pero no deja una linea notable como lo hacia el anterior elemento


¿Para qué sirve: <ol> </ol> ? 

Sirve para crear una lista ordenada, el contenido de esta
lista deben estar dentro de etiquetas <li> </li>, por ejemplo:
<ol>
  <li>Arroz</li>
  <li>Pollo</li>
  <li>etc...</li>
</ol>


¿Para qué sirve: <ul> </ul> ? 

Sirve para crear una lista des-ordenada, el contenido de esta 
lista debe estar dentro de etiquetas <li> </li, por ejemplo:
<ul>
  <li>Arroz</li>
  <li>Pollo</li>
  <li>etc...</li>
</ul>


¿Para qué sirve: <li> </li> ?

Sirve para definir el contenido de un elemento de una lista, sea ordenada o no.


¿Para qué sirve: <div> </div> ?

Etiqueta común utilizada para crear un contenedor genérico. 
No aporta semántica.
